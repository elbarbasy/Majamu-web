# Cashier UI Specification

## Layout
- Tablet Landscape First
- Realtime Order Board

## Main Sections
- New Orders
- Processing Orders
- Ready Orders
- Completed Orders

## Order Card
- Order Number
- Table Number
- Customer Name (optional)
- Product List
- Total Amount
- Payment Method
- Current Status

## Actions
- Confirm Payment
- Start Preparing
- Mark Ready
- Complete Order

## Additional Features
- Shift Notes
- Realtime Notifications
- Stock Out Toggle
