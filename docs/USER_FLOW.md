# Majamu User Flow

## Customer Flow
1. Scan QR Meja
2. Sistem membaca nomor meja otomatis
3. Pelanggan masuk ke halaman menu
4. Gunakan filter dan quiz rekomendasi sesuai PRD
5. Pilih produk
6. Tambah ke keranjang
7. Checkout
8. Isi WhatsApp dan data yang diperlukan
9. Pilih metode pembayaran
10. Pesanan masuk ke dashboard kasir
11. Pantau status pesanan
12. Riwayat tersimpan di perangkat pelanggan

## Cashier Flow
1. Login
2. Melihat order masuk
3. Konfirmasi pembayaran tunai
4. Mulai racik
5. Tandai siap diambil
6. Tandai selesai
7. Catatan shift

## Owner Flow
1. Login
2. Melihat dashboard
3. Mengelola menu
4. Mengelola filter chip
5. Mengelola produk
6. Melihat laporan
7. Mengelola pengaturan
