## UX Rules

- Mobile First
- No Customer Login
- No Bottom Navigation
- Hamburger/Menu hanya untuk informasi dan riwayat
- Quiz tampil sebagai Full Screen Bottom Sheet
- Filter selalu horizontal scroll satu baris
