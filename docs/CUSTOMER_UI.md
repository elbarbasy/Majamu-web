# Customer UI Specification

## Header
- Hamburger Menu (kiri)
- Logo Majamu
- Search Menu

## Hamburger Menu
- Beranda
- Riwayat Pesanan
- Tentang Majamu
- Kontak

## Homepage
1. Banner Promo
2. Quiz Rekomendasi Jamu (mengikuti pertanyaan pada PRD)
3. 11 Filter Cepat sesuai PRD
4. Kategori Produk
5. Produk Populer
6. Semua Produk
7. Floating Cart

## Product Card
- Foto Produk
- Nama Produk
- Harga
- Badge populer (opsional)
- Tombol tambah ke keranjang

## Cart
- Daftar Produk
- Catatan Pesanan
- Ringkasan Harga
- Checkout

## Checkout
- Nama Pelanggan
- Nomor WhatsApp
- Nomor Meja (otomatis dari QR)
- Catatan Pesanan
- Metode Pembayaran

## Riwayat Pesanan
- Disimpan berdasarkan device dan nomor WhatsApp
- Tidak memerlukan login pelanggan

## Tracking Pesanan
Status:
- Pending
- Confirmed
- Preparing
- Ready
- Completed
- Cancelled
