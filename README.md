# Majamu Web

Platform pemesanan menu berbasis QR untuk restoran dan kedai jamu.

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase
- Midtrans
- Fonnte
- Vercel

## MVP Features

### Pelanggan
- Scan QR Meja
- Lihat Menu
- Cari Produk
- Filter Produk
- Keranjang
- Checkout
- Tracking Pesanan
- Riwayat Pesanan

### Kasir
- Login
- Kelola Pesanan
- Update Status
- Shift Notes

### Owner
- Dashboard
- Produk
- Kategori
- Banner
- QR Meja
- Laporan
- Pengaturan
